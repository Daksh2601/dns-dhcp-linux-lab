# dns-dhcp-linux-lab
Deployed BIND9 DNS and ISC-DHCP Server on Ubuntu Server 24.04 LTS inside VirtualBox. Configured authoritative DNS zone (lab.local), dynamic IP allocation (192.168.56.100–200), static interface via Netplan, and remote SSH management. Built as part of a hands-on IT/networking portfolio targeting IT Administrator and Junior Network Engineer roles.
